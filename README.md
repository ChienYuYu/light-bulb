## 使用技術、套件

|  技術/套件   | 簡述  |
| :----- | :----- |
| Vue Cli   | 以 Vue.js 開發 SPA 網站 |
| VueRouter  | 設定路由 |
| Vuex  | 數據共享 |
| Bootstrap 5  | 格線系統使用、開發 UI 介面 |
| AOS  | 元件進場動畫 |
| Swiper  | 圖片輪播 |
| vue-sweetalert2  | 彈出提示窗 |
